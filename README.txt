This is a personal project made specifically for Romanian World of Warcaft players, to be more precise the guild that I raid into Dumbrava Minunata.

This bot will take input from users on Discord such as a grade and a comment and will store it into a MongoDB database.

We do this using the command !review <insert raider.io link> and then we will recieve a DM from the bot where we have to insert a grade and a comment

Whenever someone wants to see if a player is good enough or bad for a certain key that you would like to take him into (usually pugs who are random people that queue to your dungeon)
you would have to type !check <insert raider.io link> and then you will recieve a DM on Discord with all the comments that the player has and also an avarage of all the grades that he recieved from the reviews.
